# 🚀 My Portfolio

A clean, dark-mode developer portfolio built with **React + Vite**.

## 🛠 Tech Stack
- React 18 + Vite
- Google Fonts (Cormorant Garamond + Bricolage Grotesque + IBM Plex Mono)
- Pure CSS-in-JS — no extra libraries needed

## 🚀 Run Locally

```bash
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)

## ✏️ Customize
Edit `src/App.jsx` — all your info is at the top:
- **Name** → search `Your Name`
- **Projects** → edit the `projects` array
- **Skills** → edit the `skillGroups` array
- **Email & links** → edit the contact section

## 📦 Build for Production
```bash
npm run build
```

## 🌐 Deploy Free on Vercel
1. Push to GitHub
2. Go to [vercel.com](https://vercel.com) → Import project → Deploy ✅
